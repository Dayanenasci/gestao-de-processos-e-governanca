# Gestão de Processos e Governança

Projeto desenvolvido para simular organização e controle de processos internos, com foco em rastreabilidade, acompanhamento e melhoria contínua.

## Objetivo

Demonstrar estruturação de fluxo, controle de demandas e definição de indicadores de acompanhamento.

## Estrutura

- mapeamento → descrição de fluxo do processo
- controles → controle de chamados e solicitações
- indicadores → definição de KPIs

## Conceitos aplicados

- Governança
- Organização de processos

## Automação e IA

O processo foi projetado considerando automação com Power Automate e apoio de IA para classificação, comunicação e análise de recorrência, garantindo eficiência operacional e governança.
